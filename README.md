Ferns N Petals Sales Analysis Dashboard 📌 Project Overview This project is an interactive Sales Analysis Dashboard built in Microsoft Excel to analyze sales performance, customer behavior, product trends, and occasion-based revenue insights for Ferns N Petals (FNP). The dashboard enables stakeholders to track KPIs and make data-driven business decisions through dynamic visualizations and interactive filters. 🎯 Business Objectives Monitor overall sales performance. Analyze monthly revenue trends. Identify top-performing product categories. Evaluate occasion-wise sales contribution. Understand customer purchasing patterns.
